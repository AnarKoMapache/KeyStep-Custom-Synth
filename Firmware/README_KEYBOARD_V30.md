# KeyStep V30 - Reface DX-ish FM Lead + Arpegiador

Este firmware usa la matriz real del KeyStep original:

- Select: PE8..PE15, activo en bajo
- Read: PD8..PD15, activo en bajo
- Audio: Pitch CV / DAC1 / PA4
- Mod CV / DAC2 / PA5 queda centrado

## Comportamiento

- 1 tecla: mono lead FM con portamento/glide
- 2+ teclas: arpegiador UP interno
- Sonido: FM digital tipo Yamaha Reface DX/DX-ish, con ataque brillante y cuerpo más limpio que el Moog-ish

No conecta audífonos directo al CV. Usar mixer, interfaz, entrada de sinte o Crave con volumen bajo.
