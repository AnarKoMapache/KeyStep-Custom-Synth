# KeyStep V40 - Velocity/Accent Mod CV

Base: V39 Mod CV performance.

Cambios:
- Audio principal por Mod CV/DAC2/PA5, según pruebas reales.
- Mantiene 5 motores, octava, brillo, vibrato y hold/latch.
- Agrega velocity/acento aproximado usando doble contacto A/B del teclado.
- Escaneo más rápido para medir el tiempo entre contactos.

Controles ocultos:
- tecla más grave + 1..5: motores
- tecla más grave + 6/7/8: octava abajo/normal/arriba
- tecla más grave + 9/10/11: oscuro/normal/brillante
- tecla más grave + 12/13/14: vibrato off/normal/profundo
- tecla más grave + 15: hold/latch ON/OFF
- tecla más grave + 16: velocity/accent OFF
- tecla más grave + 17: velocity/accent ON

Nota: si las teclas no muestran mucha diferencia de dinámica, dejar velocity ON como acento/brillo y ajustar umbrales en V41.
