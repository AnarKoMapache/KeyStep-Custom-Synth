# KeyStep V34 MicroFreak-ish No Arp

Firmware experimental para KeyStep original / STM32F103VCT6.

- Matriz real: PE8-PE15 seleccionan, PD8-PD15 leen.
- Salida audio: Pitch CV / DAC1 / PA4.
- Sin arpegiador.
- Monofónico.
- Motor digital tipo MicroFreak-ish: wavetable morph, FM suave, wavefolding y movimiento lento interno.

No conectar audífonos directo al CV. Usar mixer, interfaz, entrada de sinte o módulo con volumen bajo.
