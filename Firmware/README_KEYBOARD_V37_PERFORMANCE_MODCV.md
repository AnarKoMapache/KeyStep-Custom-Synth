# KeyStep V37 Performance + Mod CV

- Audio principal: Pitch CV / DAC1 / PA4.
- Mod CV / DAC2 / PA5: salida de control lenta, mezcla de envolvente + LFO.
- Sin arpegiador.
- Monofonico.
- 5 motores heredados del V35/V36.

## Comandos ocultos

Mantener la tecla mas grave y tocar:

- +1: MicroFreak-ish
- +2: Yamaha FM
- +3: Moog lead
- +4: Acid bass
- +5: Digital bell/pad
- +6: octava abajo
- +7: octava normal
- +8: octava arriba
- +9: timbre oscuro
- +10: timbre normal
- +11: timbre brillante
- +12: vibrato off
- +13: vibrato normal
- +14: vibrato profundo

No conectar audifonos directo al CV. Usar mixer/interfaz/sinte con volumen bajo.
