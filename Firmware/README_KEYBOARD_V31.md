# KeyStep V31 - Yamaha / Reface DX-ish FM Mono (sin arpegiador)

- Teclado real: PE8-PE15 seleccionan, PD8-PD15 leen, activo en bajo.
- Salida: Pitch CV / DAC1 / PA4.
- Sin arpegiador: si mantienes varias teclas, queda en prioridad mono.
- Sonido: FM más digital/brillante, tipo Yamaha/Reface DX-ish.

No conectar audífonos directo al CV. Usar entrada de sinte, mixer o interfaz con volumen bajo.
